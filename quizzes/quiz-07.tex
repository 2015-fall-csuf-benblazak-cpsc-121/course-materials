\section{Question}
\standards{
  \standard{functions}
}

Assuming that all necessary headers have been included and all required symbols
are in the standard namespace, do the following:

\begin{itemize}

  \item Write the prototype and definition for a function named
    \mintinline{cpp}{average} that accepts three \mintinline{cpp}{int}s and
    returns their average (arithmetic mean) as a \mintinline{cpp}{double}.
    \textAnswer{\inputminted{cpp}{\codeFilePrefix.average}}
    \vfill

  \item Write the prototype and definition for a function named
    \mintinline{cpp}{set} that accepts two \mintinline{cpp}{int}s, the first
    passed by value and the second passed by reference, sets the value of the
    second to twice the value of the first, and returns \mintinline{cpp}{void}.
    \textAnswer{\inputminted{cpp}{\codeFilePrefix.set}}
    \vfill

  \item Write a short \mintinline{cpp}{main()} and indicate its output.  Your
    \mintinline{cpp}{main()} must begin with
    \mintinline{cpp}{int a = 3, b = 2, c = 1;} and accomplish the following:
    \begin{itemize}
      \item Output the value of \mintinline{cpp}{average} called with the
        arguments \mintinline{cpp}{a}, \mintinline{cpp}{b}, and
        \mintinline{cpp}{c}, followed by a newline.
      \item Call \mintinline{cpp}{set} with \mintinline{cpp}{a} as the first
        argument and \mintinline{cpp}{b} as the second.
      \item Output the values of \mintinline{cpp}{a}, \mintinline{cpp}{b}, and
        \mintinline{cpp}{c}, separated by whitespace.
    \end{itemize}
    \textAnswer{\showCodeLeft{functions}{main}}
    \vfill

\end{itemize}
